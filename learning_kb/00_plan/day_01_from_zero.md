## Steps

1. 安装 Godot 4.x stable（官网下载，Windows 版本）。
2. 打开 Project Manager，新建项目到 `godot_study/projects/first_game`。
3. 创建并保存 `Main` 场景（先可运行空场景）。
4. 在 Project Settings 预创建 Input Map 条目。
5. 若仍有时间，再创建 `Player`（`CharacterBody2D`）和最小移动脚本。
