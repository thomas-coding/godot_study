# Week 01 Bootstrap Plan

适用对象：Godot 完全新手（有 C/C++ 工程背景）。

## Week 01 Milestone

- 完成安装与第一个可运行 2D 原型。
- 建立固定学习 SOP（输入 -> 实作 -> 复盘）。
- 形成第一批术语卡（15-25 条）。

## Lesson Plan (2h x 5, default)

1. 第1课：安装、创建项目、Input Map、最小运行验证。
2. 第2课：`CharacterBody2D` 最小可玩（左右移动 + 跳跃 + 地面碰撞）。
3. 第3课：场景复用（PackedScene）、可收集物、碰撞事件。
4. 第4课：UI 基础（开始/暂停/重开），最小游戏流程。
5. 第5课：导出前的玩法稳定化与回归检查。

说明：按“课次”推进，不绑定自然日。一天可学多课，也可不学习。

## 本周验收标准

- 能独立解释 Scene/Node/Signal 的基本关系。
- 能从英文文档定位所需类和方法。
- 能输出一份英文简短日报（5-8 行）并附术语清单。
