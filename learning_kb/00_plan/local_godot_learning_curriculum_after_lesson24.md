# Local Godot Learning Curriculum After Lesson 24

Last Updated: 2026-07-01
Scope: 本地 Godot 游戏制作学习，不以 Steam 发布为目标
Status: Planning reference for future mentor prep

## 规划原则

- 目标是学习 Godot 制作能力，不是把当前游戏内容无限做大。
- 后续课程按“能力覆盖”组织，而不是按“游戏内容堆料”组织。
- 每课仍保持 2h runbook 形式，并要求有最小可运行/可验证结果。
- 第25课以后备课时优先参考本文件；具体课次仍以 `lesson_queue.md` 中已生成 runbook 为准。

## 当前基线

- 第1课到第26课已完成：Godot 2D 核心系统、vertical slice 主链路、Boss encounter、输入边界。
- 第27课已备课：UI/菜单系统与状态流。
- 第28课到第30课已备课：反馈时序、音频系统、Resource 深化。
- 第31课以后尚未生成正式 runbook。

## 必修课程规划（第25课到第36课）

### 第25课：Boss 战补强

- 学习重点：复杂对象状态、伤害判定、阶段切换、门控边界。
- 能力目标：理解一个复杂玩法对象如何拆分状态、输入、碰撞、反馈与结算。
- 备注：重点不是“做大 Boss”，而是学习复杂对象组织方式。

### 第26课：输入与交互系统整理

- 学习重点：`InputMap`、暂停态输入、UI 输入、快捷键冲突、改键基础。
- 能力目标：能解释 gameplay input 与 UI input 的边界，并能处理常见冲突。

### 第27课：UI/菜单系统

- 学习重点：`Control` 布局、主菜单、暂停菜单、设置菜单、结算界面之间的关系。
- 能力目标：能组织一个小项目常见 UI 流程，并避免游戏状态与 UI 状态互相污染。

### 第28课：Animation / Tween / Timer

- 学习重点：常见反馈实现方式，受击闪烁、弹窗、过场、冷却、延迟执行。
- 能力目标：理解一次性反馈、持续状态、定时事件之间的实现差异。

### 第29课：音频系统基础

- 学习重点：音效、背景音乐、Audio Bus、音量设置、场景切换时音频状态。
- 能力目标：能接入并管理基础音频反馈，理解音量设置和总线分层。

### 第30课：Resource 与数据驱动深化

- 学习重点：配置敌人、关卡、奖励、难度；资源共享、复制、运行时修改边界。
- 能力目标：能判断哪些数据适合放进 `Resource`，并避免误改共享资源。

### 第31课：场景组织与复用

- 学习重点：如何拆 scene、如何复用 reusable scenes、如何避免 main scene 膨胀。
- 能力目标：建立小项目可维护的 scene 组织习惯。

### 第32课：存档与设置完整化

- 学习重点：存档结构、默认值、版本字段、坏档 fallback、本地设置持久化。
- 能力目标：能设计一个小型本地游戏的可靠存档/设置结构。

### 第33课：调试与排错专题

- 学习重点：Debugger、Output、断点、远程场景树、最小复现、日志分级。
- 能力目标：形成独立定位问题的工作流，而不是只靠猜。

### 第34课：性能与稳定性专题

- 学习重点：Profiler、帧时间、对象数量、碰撞成本、场景切换/重开回归。
- 能力目标：能用证据判断是否有性能问题，并维护基础稳定性回归清单。

### 第35课：本地导出与运行包验证

- 学习重点：Windows 导出、资源包含、导出包启动、本地 build 与编辑器运行差异。
- 能力目标：能独立产出一个本地可运行包并验证主链路。

### 第36课：小项目收口复盘

- 学习重点：把一个 Godot 项目从原型整理成可交接、可继续开发的状态。
- 能力目标：总结架构、坑点、检查表，并形成下一项目可复用经验。

## 选修课程规划（第37课到第42课）

### 第37课：Shader / 2D 视觉效果基础

- 学习重点：CanvasItem shader、简单描边/闪烁/溶解/颜色替换。
- 能力目标：理解 Godot 2D shader 的基本输入输出和可控视觉反馈。

### 第38课：TileMap / 关卡编辑工作流

- 学习重点：TileSet、TileMapLayer、碰撞、场景化 tile、关卡搭建效率。
- 能力目标：能用 TileMap 建立更适合扩展的 2D 关卡工作流。

### 第39课：手柄输入与多输入设备

- 学习重点：键盘/手柄输入抽象、按键提示、设备差异、菜单导航。
- 能力目标：理解多输入设备适配的基本边界。

### 第40课：简单 AI / 寻路 / 状态行为树入门

- 学习重点：有限状态、追踪/巡逻/攻击选择、NavigationAgent2D 入门。
- 能力目标：能为敌人建立更清晰的行为组织方式。

### 第41课：自动化测试或半自动回归

- 学习重点：可重复回归清单、命令行运行、基础 smoke test、结果记录。
- 能力目标：把手工测试经验转成可复用的稳定性流程。

### 第42课：Godot 插件 / Editor 工具脚本入门

- 学习重点：`@tool`、EditorPlugin 基础、定制小工具、批量检查。
- 能力目标：理解 Godot 编辑器可扩展能力，能为项目制作轻量工具。

## 删减或降级内容

- Steam 商店流程：不作为本地学习主线。
- Steamworks SDK：不进入必修。
- 成就、云存档、排行榜：作为远期兴趣内容，不纳入第25-42课。
- 商店页素材与上线流程：不纳入本地学习路线。
- 多平台发布策略：仅保留 Windows 本地导出。

## 总体结论

- 必修建议做到第36课左右，覆盖本地 Godot 游戏制作关键能力。
- 加上选修后，到第42课可以形成较完整的 Godot 2D 学习闭环。
- 后续导师线备课应优先把第31课到第33课生成正式 runbook，并保持至少 3 课缓冲。
