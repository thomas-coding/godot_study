## Key Understanding

- `game_balance_config.gd` defines the data shape.
- `.tres` files store concrete Resource values.
- `preload("res://configs/balance_default.tres")` provides a code-level default.
- `@export` exposes the Resource reference in Inspector, so a scene can override which `.tres` file a node reads.
