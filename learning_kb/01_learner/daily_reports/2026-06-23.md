## Learner Understanding Confirmed

- `projectile.tscn` is a reusable object template, not a new level.
- `projectile.gd` defines behavior; `instantiate()` creates runtime instances from the scene template.
- `_physics_process(delta)` runs once per physics frame while the node is active and unpaused.
- `delta` is elapsed seconds since the previous physics tick, used for frame-rate-independent movement and lifetime.
- `Vector2(20, -8)` is a spawn offset from the enemy, separate from projectile flight direction.
- `add_child(projectile)` is the point where the runtime instance enters the active scene tree and begins `_ready()`, `_physics_process()`, drawing, and collision detection.
