# QA087 - Why plan lessons by learning capability instead of game features?

Question:
- 为什么第25课以后要按学习能力规划，而不是继续按游戏内容扩展？

Version Scope: 4.6
Last Verified: 2026-06-30
Compatibility: unchanged

Answer (teaching-ready):
- 因为当前目标是学习 Godot，不是把这个游戏本身做大。游戏功能只是练习载体。
- 每节课应该回答一个能力问题：
  - 我学会了哪类 Godot 系统？
  - 这个能力下个项目还能不能复用？
  - 这个功能是否只是为了堆内容？
- 所以第25课 Boss 补强的真正目标不是“做更厉害的 Boss”，而是学习复杂玩法对象如何组织状态、碰撞、反馈和重置边界。

Evidence:
- `learning_kb/00_plan/local_godot_learning_curriculum_after_lesson24.md`
- `learning_kb/02_mentor/modules/M22_learning_first_local_godot_curriculum.md`

Validation:
- 每个 runbook 都应有一句学习目标，而不只是功能目标。
