# Graphics Foundations (Study Plan)

目标：为后续游戏开发与性能优化建立图形学基础，不追求学术化推导，强调“能解释 + 能落地”。

## Module 1: Coordinate and Transform Basics

- Vector2/Vector3
- Local vs global transform
- Rotation and radians/degrees conversion

Status: `planned`

## Module 2: Camera and View

- Camera2D follow and smoothing
- Jitter diagnosis and frame-update alignment

Status: `planned`

## Module 3: Rendering and Frame Pipeline

- Frame time, FPS, update loop basics
- CPU update vs rendering cost intuition

Status: `planned`

## Module 4: Practical Math for Gameplay

- Interpolation (`lerp`), easing basics
- Velocity, acceleration, gravity in gameplay code

Status: `planned`

## Evidence Strategy

- Prefer Godot official docs + reproducible mini demos.
- 每个模块至少附一个“可运行示例”和一个“排错清单”。
