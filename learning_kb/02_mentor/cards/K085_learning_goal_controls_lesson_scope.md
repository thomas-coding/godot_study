# K085 - Learning goal controls lesson scope

Status: Working
Version Scope: 4.6
Last Verified: 2026-06-30
Compatibility: unchanged

Conclusion:
- If the goal is learning Godot, each feature lesson must name the transferable capability it teaches. Otherwise, the course drifts into content production instead of skill acquisition.

Evidence:
- `learning_kb/00_plan/local_godot_learning_curriculum_after_lesson24.md`
- `learning_kb/02_mentor/modules/M22_learning_first_local_godot_curriculum.md`

Teaching Use:
- Before a new runbook, state the learning objective first, then the game change that carries it.
